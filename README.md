# BARBER NEXUS — ランディングページ

バーバーサロン「BARBER NEXUS」のランディングページです。

## デモ

**[https://imtyidsay.github.io/barber-lp/](https://imtyidsay.github.io/barber-lp/)**

## 概要

- 純粋な HTML / CSS で構築（フレームワーク不使用）
- レスポンシブ対応（モバイル・タブレット・デスクトップ）
- スクロールアニメーション（Intersection Observer API）
- ハンバーガーメニュー（スマートフォン向け）

## ページ構成

| セクション | 内容 |
|---|---|
| Hero | キャッチコピー・Web予約ボタン |
| Services | サービス一覧 |
| Gallery | スタイルギャラリー |
| Stylists | スタイリスト紹介 |
| Features | お店のこだわり |
| Price | 料金表 |
| FAQ | よくある質問 |
| Access | アクセス・地図 |
| Reserve | Web予約フォーム |
